Summary: Latitude vs. X

This project is an exploration of the impact of latitude on four different weather measurements using a random list of cities around the globe. First a list of six hundred valid latitude/longitude pairs are created and then run through the citipy module to find the closest city to each geographic point. Then the list of cities is used to pull weather data from the OpenWeatherMap API. Since some cities are not included on the OpenWeatherMap website, this process is repeated, if necessary, until a minimum of 500 cities are included in the dataset.

The weather measurements analyzed were temperature, wind speed, humidity, and cloudiness. Of the four, only temperature showed any significant correlation with the latitude measurement, with wind speed showing the largest outliers at the more extreme latitudes, and humidity and cloudiness showing no significant variation with latitude. The four generated charts are shown under the Visualizations header on this page, each leading to an individual analysis page. The full dataset is also available on this site.
